# 🥗 健康外卖推荐

一个纯前端网页：**定位附近 → 按营养成分推荐健康外卖 → 一键跳转美团 / 饿了么搜索**。
专治"只点碳水"，帮人吃得更均衡。

## 功能

- 📍 **真实定位**：浏览器获取位置，用 OpenStreetMap 反查城市（带容错）。
- 🥗 **营养推荐**：内置常见健康菜品营养库（热量 / 蛋白 / 碳水 / 脂肪），按目标智能打分排序。
- 🎯 **三种目标**：均衡饮食 / 减脂控碳 / 增肌高蛋白，一键切换。
- 💰 **预估价格**：每道菜给出参考价格区间。
- 🔗 **一键跳转**：点按钮带菜名去美团 / 饿了么搜索下单。
- ⚠️ **碳水预警**：碳水偏高的菜会标黄提醒，纠正"只点碳水"的习惯。

## 在线使用

部署在 GitHub Pages，打开网址即可（手机端定位最准）。

## 本地运行

用 Chrome / Edge 直接打开 `index.html` 即可预览。
> 本地 `file://` 打开时定位可能受限，建议用 GitHub Pages 的 https 网址使用。

## 自定义菜品

编辑 `index.html` 中的 `DISHES` 数组，按格式增改菜品：

```js
{ name: "菜名", cat: "分类", kcal: 320, protein: 35, carb: 18, fat: 11, price: [22, 32], tags: ["高蛋白"], kw: "搜索关键词" }
```

字段说明：`kcal` 热量(千卡)，`protein/carb/fat` 蛋白/碳水/脂肪(克)，`price` 价格区间，`kw` 跳转平台时的搜索词。

## 说明

美团 / 饿了么无对外免费接口，纯前端无法实时抓取附近真实店铺菜单与价格。
因此推荐基于内置营养库、价格为参考值，跳转按钮会带菜名去平台**搜索页**下单。

## 技术栈

纯静态：HTML + CSS + 原生 JavaScript，无需后端，无依赖。
